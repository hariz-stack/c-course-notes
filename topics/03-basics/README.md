# C Basics: printf & Comments

## Key Ideas

- `#include <stdio.h>` gives access to `printf()`
- `main()` is the program entry point
- `return 0;` indicates success
- `\n` creates a newline

## Comments

- Single-line: `// comment`
- Multi-line: `/* comment */`

## printf() Examples

Use `printf()` to display text and variables to the console.

```c
printf("Text here\n");
printf("Value: %d\n", 42);
```
