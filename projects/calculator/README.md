# Calculator Project

## Goal

Build a simple calculator that performs basic arithmetic operations (+, -, *, /).

## Concepts Used

- User input
- Switch statements
- Error handling (division by zero)
- Conditional logic

## How It Works

1. Ask user for first number
2. Ask for an operator (+, -, *, /)
3. Ask for second number
4. Use `switch` to determine operation
5. Display result
6. Handle division by zero

## Compile & Run

```bash
gcc -o calculator calculator.c
./calculator
```
