# Weight Converter Project

## Goal

Convert weight between kilograms and pounds using conditional logic.

## Concepts Used

- User choice with menu
- Conditional statements (if/else)
- Mathematical conversion formulas
- Formatted output

## Conversions

- kg → lb: `lb = kg × 2.20462`
- lb → kg: `kg = lb ÷ 2.20462`

## How It Works

1. Display menu with two options
2. Ask user to choose
3. Based on choice, perform appropriate conversion
4. Display result

## Compile & Run

```bash
gcc -o weight_converter weight_converter.c
./weight_converter
```
