# Shopping Cart Project

## Goal

Read item name, price, and quantity from the user, then calculate and display the total cost.

## Concepts Used

- User input with `scanf()` and `fgets()`
- String handling
- Arithmetic operations
- `printf()` formatting

## How It Works

1. Ask user for item name
2. Ask for price per item
3. Ask for quantity
4. Calculate total: `total = price × quantity`
5. Display results

## Compile & Run

```bash
gcc -o shopping_cart shopping_cart.c
./shopping_cart
```
