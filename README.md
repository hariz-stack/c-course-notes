## How to Use

1. **Learning Concepts:** Browse the `topics/` folder for structured lessons with code examples
2. **Building Projects:** Check the `projects/` folder for complete, runnable implementations
3. **Compiling & Running:**
   ```bash
   gcc -o program program.c
   ./program
   ```

## Topics Covered

- Introduction to C
- Setup & Compilation (gcc/clang)
- Basics (printf, comments, variables)
- Data Types (int, float, double, char, bool)
- Operators (arithmetic, logical, ternary)
- Control Flow (if/else, switch, nested conditionals)
- Functions (definition, return, scope, prototypes)
- Loops (while, for, break, continue)
- Arrays (1D, 2D, arrays of strings)
- Advanced Types (typedef, enum, struct)
- Pointers (addresses, dereferencing)
- File I/O (reading/writing files)
- Memory Management (malloc, calloc, realloc)
